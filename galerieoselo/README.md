##  Installation

1. Cloner le projet dans `htdocs`
2. Créer la base de données `oselo_gallery` via phpMyAdmin
3. Importer les tables SQL (voir `db.php`)
4. Lancer `http://localhost/galerie-oselo` dans votre navigateur
